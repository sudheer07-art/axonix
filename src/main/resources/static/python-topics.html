Introduction to Python

Python is a high-level, interpreted, dynamically typed programming language created by Guido van Rossum. It is known for its simplicity, readability, and vast ecosystem.

Python is widely used in Web Development, Data Science, AI, Machine Learning, Automation, and DevOps.

Python Syntax

Python uses indentation instead of braces.

print("Hello Python")

Variables

name = "Sudheer"

Introduction to JavaScript

JavaScript is a high-level, interpreted programming language used to make web pages interactive. It runs in the browser and on servers (Node.js).

Used for dynamic UI, form validation, animations, APIs, and full-stack development.

JavaScript Syntax

console.log("Hello JavaScript");

Variables

JavaScript supports var, let, and const. 

let x = 10;
const name = "Sudheer";

Data Types

  • Number
  • String
  • Boolean
  • Undefined
  • Null
  • Object
  • Symbol

Operators

Arithmetic, comparison, logical, assignment, ternary operators.

Control Flow

if, else if, switch statements control execution.

Loops

for, while, do-while, forEach, for...of

Functions

function add(a,b){
    return a+b;
}

Arrow functions were introduced in ES6.

Scope

Global scope, function scope, block scope.

Arrays

Arrays store ordered collections of values.

Objects

Objects store key-value pairs.

Strings

String methods include slice, substring, replace, split.

DOM (Document Object Model)

DOM allows JavaScript to interact with HTML elements.

Events

Events respond to user actions like click, submit, hover.

ES6+ Features

let/const, arrow functions, destructuring, spread operator, modules.

Asynchronous JavaScript

Callbacks, Promises, async/await handle async operations.

Error Handling

try {
  let x = y;
} catch(e) {
  console.log(e);
}

Web Storage

localStorage and sessionStorage store data in browser.

JavaScript Best Practices

Use const/let, avoid global variables, modular code, clean naming.

Introduction to C

C is a procedural programming language developed by Dennis Ritchie. It is fast, memory-efficient, and forms the foundation of many modern languages.

Used in Operating Systems, Embedded Systems, Compilers, Drivers.

C Program Structure

#include <stdio.h>

int main(){
    printf("Hello C");
    return 0;
}

Every C program starts execution from the main() function.

Data Types

  • int
  • float
  • double
  • char

Control Statements

if, else, switch, for, while, do-while

Arrays

Collection of same data types stored in contiguous memory.

Pointers

Pointers store memory addresses and allow dynamic memory access.

Functions

Functions help in code reuse and modular programming.

Structures

Structures allow grouping of different data types.

File Handling

Introduction to HTML

HTML (HyperText Markup Language) is the backbone of the web. It defines the structure and content of web pages.

HTML Document Structure

<!DOCTYPE html>
<html>
<head></head>
<body></body>
</html>

This structure is mandatory for every HTML document.

HTML Elements & Tags

Elements consist of opening tag, content, and closing tag.

HTML Attributes

Attributes provide additional information about elements like id, class, src, href.

Text Formatting

Bold, italic, underline, strong, emphasis, mark, small.

Links

Anchor tag is used to link pages and external resources.

Images

Images enhance visual presentation of content.

Lists

Ordered, unordered, and description lists.

Tables

Used to display data in rows and columns.

Forms

Forms collect user input using input, textarea, select, button.

Semantic HTML

Semantic elements describe meaning and improve SEO and accessibility.

Audio & Video

HTML5 supports native audio and video playback.

Iframes

Used to embed other webpages or applications.

Meta Tags

Meta tags define page metadata like charset and viewport.

SEO Basics

Proper headings, meta tags, alt attributes improve SEO.

Accessibility

Accessibility ensures websites are usable for everyone.

HTML Best Practices

Use semantic tags, proper indentation, valid markup.
